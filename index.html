What is Patchbot?

Patchbot is a Self Service Automated Patching Service created with existing IT infrastructure. Patchbot allows customers to request desired maintenance windows and subscribe existing hosts to their Patchbot automated patching service.

Infrastructure as Code

Patchbot is built as code across existing IT services.
